About

Orientering fixture browsing

Key features

Fixture information

  • Linked map and grid/list view of fixturess
  • Event location from grid references, postcode, Google search on area, or nominal club location
  • Routegadget maps for the area
  • Google, Ordnance survey and What3Words maps
  • Add event to google calander (Google)
  • Support for displaying event series and multi-day events (JK/British etc)

User functionality

  • Mark 'liked' events of interest

Search/Filtering

Search/Filtering Fixture filtering based on:
  • Distance from postcode/grade
  • Day of week
  • User 'liked' events
  • Search string of title/club/area

Entries

  • Enter events registered with Racesignup, Fabian4 and SIEntries

Application

  • Optimised for desktop and mobile
  • Fixture data avalaible offline

Additional events

  • Additional event sources (MBO/Army/Mountain marathons)
